# mission-control-server

Intermedial [BLATT 3000](https://www.blatt3000.de) lecture at UNM Festival 2016, Aarhus.

## Requirements

* Node 6.3.1

## Installation

```
git clone git@github.com:marmorkuchen-net/mission-control-server.git
cd mission-control-server
npm install
```

## Usage

```
npm start
```
